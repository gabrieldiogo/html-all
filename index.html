(I) lista ordenadas

O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

(IV) image

Para inserir uma imagem no HTML basta utilizar a tag img com o atributo src. Ou seja: o atributo src, ou source, vai conter a url da imagem que será inserida.

(V) links

No HTML, os links são definidos pela tag a. Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link.
